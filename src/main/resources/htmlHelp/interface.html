Jip 1.0 main interface

A: main toolbar, including Home, Patient, Inpatient, Surgical, Medication & Help

B: sub toolbar for each sub panel, including the actions Add, Search, Update, Delete, Print & Close

C: record detail form to be used by search, add or update action

D: record list table, double click the single record to update

E: additional toolbar for patient view. After select the record from patient table, you can edit inpatient, surgical & medication details, and also release the patient, get the report
