Jip 1.0 patient page
  1. How to add new patient
    click Add button from the main toolbar, fill new patient details and click Add action button

  2. How to search patient
    click Search button from the main toolbar, fill the search criteria and click Search action button

  3. How to update patient
    select one single record from the table, click update button from the main toolbar, update the details and click Update action button

  4. How to delete patient
    select one single record from the table, click delete button from the main toolbar

  5. How to modify the inpatient, surgical & medication details of the patient
    select one single record from the table, click Edit inpatient, surgical & medication button in the button

  6. How to print the patient report
    select one single record from the table, click Report button in the button
