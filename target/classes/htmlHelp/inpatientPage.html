Jip 1.0 inpatient page
  1. How to add new inpatient
    click Add button from the main toolbar, fill new inpatient details and click Add action button

  2. How to search inpatient
    click Search button from the main toolbar, fill the search criteria and click Search action button

  3. How to update inpatient
    select one single record from the table, click update button from the main toolbar, update the details and click Update action button

  4. How to delete inpatient
    select one single record from the table, click delete button from the main toolbar
